Welcome to the MagicPrompt Tab

This extension helps you generate magic prompts with ease.

  • Generate Detailed Prompts

    Generate a rewritten prompt with more detail directly in SwarmUI.

  • Supports Multiple LLMs

    Supports local Ollama LLMs, OpenAI, and Anthropic (requires API keys).

  • Easy-to-Use Interface

    Send the rewritten prompts to the Generate tab with a simple click.

    Welcome to the UniversalTab

    This tab allows you to add any webpage to your SwarmUI interface.

  • Create an iFrame Tab

    Display any webpage in a new tab within SwarmUI.

  • Flexible Integration

    Integrate external content seamlessly into your SwarmUI interface.

  • Current Limitations

    Currently supports only one extra tab, and it does not persist across refreshes or restarts.
